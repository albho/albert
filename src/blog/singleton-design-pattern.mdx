The singleton design pattern is a creational design pattern.

## Demo

<StateSingleton />

```javascript
class ApplicationState {
  constructor() {
    if (ApplicationState.instance === undefined) {
      this.state = 'OFF';
      ApplicationState.instance = this;
    }

    return ApplicationState.instance;
  }

  toggleState() {
    if (this.state === 'ON') {
      this.state = 'OFF';
    } else {
      this.state = 'ON';
    }
  }

  getState() {
    return this.state;
  }
}
```
